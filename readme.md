# EDSENSE

EDSENSE Project

## Technologies

- MongoDB
- Express.js
- React
- Node.js

## Getting Started

### Prerequisites

- MongoDB installed
- Node.js and npm installed

### Installing

1. Install NPM packages

  **Run Server**
```sh
cd server
npm install
npm start
```
    
    
  **Run Client**
```sh
cd client
npm install
npm start
```
